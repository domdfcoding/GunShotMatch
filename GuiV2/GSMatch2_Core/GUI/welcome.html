Getting Started

  • To create a new project, click the button in the toolbar above.

  • To open an existing project, click the button. You can open a recent project by clicking the arrow to the right of the icon.

  • To create a new experiment for use in a project, click the button in the toolbar above.

  • Once you have opened a project, the different files within the project can be browsed using either the tabs along the top of the Project window or the 'Project Navigator' widget to the left.

    The 'Project Navigator' widget and the tabs at the top of the Project window

  • The 'View' toolbar can be used to manipulate the view of a chromatogram, spectrum or chart.

    The 'View' toolbar

    The buttons perform the following functions:
    • Resets the view
    • Goes to the previous view
    • Enables the "Select" tool
    • Enables the "Zoom" tool for zooming the chart
    • Enables the "Pan" tool for panning the chart
    • Enables the "View Spectrum" tool for viewing a spectrum by clicking the chromatogram
    • Opens the "View Spectrum by Scan Number" tool for viewing a spectrum at the given scan number
    • Opens the "View Spectrum by RT" tool for viewing a spectrum at the given retention time
